## 这是在学习VUE的过程中，做的一个UI框架

1. 安装 happyme-ui
```
npm install happyme-ui -S
```
2. 本UI框架为个人开源产品，请谨慎使用在生产环境中
3. 不断更新，目前是做了一些常用组件（并没有element或者ant design那么全
4. 此UI框架只适用于主流浏览器（不包含IE）
